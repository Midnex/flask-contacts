# Flask contacts
![alls](screenshots/alls.jpg)

## Use
* Flask (Obvious!)
* Flask-SQLAlchemy (ORM for database)
* Flask-WTF (Generation of forms and validations)

## Install
```
python -m pip install -r requirements.txt
python migrations.py
```
## Run
```
python app.py
```
